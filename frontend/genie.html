    üí° Common Questions (HIREX)

    What file formats are supported?

    LaTeX (.tex) inputs with PDF outputs.

    Can I change the base resume?

    Yes. Replace data/samples/base_resume.tex; subsequent runs will use it.

    What does ‚ÄúHumanize‚Äù do?

    It refines fluency, tone, and clarity while keeping content truthful.

    üß© Architecture & Technology

    1. Frontend: Modular Vanilla JS (v2.0.0) with state persistence.
    2. Backend: FastAPI microservices handling AI inference & LaTeX rendering.
    3. AI Core: GPT-4o / GPT-4o-mini for inference & rewriting.
    4. Data Layer: Local JSONL analytics + adaptive caching.
    5. Security: Sandboxed compilation + key isolation.
